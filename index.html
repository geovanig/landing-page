BackEnd

Sistemas feitos com framework Spring, organizados em camadas, adapatados para Rest Api e WEB Services, com toda a parte de Basic Security. É usado banco de dados relacionais para amrmazenamento de dados, testado com JUnit, Banco de Dados H2 e Postman

FrontEnd

Sites responsivos com Bootstrap, estruturação com Angular, layouts inspirados em grandes redes sociais como LinkedIn, Twitter e são aplicadas técnicas de UX. É feito o deploy junto do BackEnd e está presente toda a comunicação do cliente com o servidor.
